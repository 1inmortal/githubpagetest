Acerca del Proyecto

Concepto

2024 - Idea inicial del sistema T-1 para detección aérea en tiempo real.

Desarrollo

Integración de Python, YOLOv8 y DJI SDK para crear una solución completa.

Optimización

Mejoras de rendimiento con aceleración GPU y procesamiento en tiempo real.

Beta Release

Sistema T-1 operativo con métricas mejoradas y estabilidad garantizada.

Proyecto T-1

Sistema de detección en tiempo real que combina la captura de video del DJI Mini 2 con procesamiento de IA usando Python y YOLOv8. Diseñado para reconocimiento de objetos en escenarios de vigilancia y monitoreo aéreo.

Stack Tecnológico

Python 3.9+ como lenguaje principal, OpenCV para procesamiento de video, YOLOv8 para detección de objetos, PyTorch para inferencia de IA, y DJI SDK para control del drone. Optimizado para procesamiento en tiempo real con aceleración GPU.
