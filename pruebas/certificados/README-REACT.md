# Visualizar Componente React

## Cómo visualizar el componente React

El componente React está ubicado en `react.js` y puede visualizarse de las siguientes maneras:

### Opción 1: Usando Vite (Recomendado)

1. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

2. **Abrir en el navegador:**
   - El servidor se abrirá automáticamente en `http://localhost:3000`
   - Navega a: `http://localhost:3000/pruebas/certificados/react.html`

### Opción 2: Acceso directo

Si el servidor ya está corriendo, simplemente abre:
```
http://localhost:3000/pruebas/certificados/react.html
```

**Nota:** Si ves un mensaje sobre la base URL `/githubpagetest/`, usa esta URL en su lugar:
```
http://localhost:3000/githubpagetest/pruebas/certificados/react.html
```

## Archivos creados

- `react.html` - Archivo HTML que monta el componente React
- `main.jsx` - Punto de entrada que renderiza el componente
- `index.css` - Estilos de Tailwind CSS
- `react.js` - Componente principal de React (ya existía)

## Dependencias instaladas

- `lucide-react` - Iconos utilizados en el componente
- `tailwindcss` - Framework CSS para estilos
- `postcss` y `autoprefixer` - Procesadores CSS

## Notas

- El componente usa Tailwind CSS para los estilos
- Los iconos provienen de `lucide-react`
- El componente está completamente funcional y listo para usar
